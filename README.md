# 🧠 Intelligent Client Solvency Prediction System

An intelligent web application that predicts customer solvency using Machine Learning and Explainable AI (SHAP). The system helps companies assess financial risk, generate a risk score, and support decision-making.

---

## 🚀 Features

- Secure authentication (JWT)
- User management
- Dataset management
- Machine Learning model training
- Training results visualization
- Customer solvency prediction
- Customer profile and payment history
- SHAP explanations
- System history

---

## 🤖 Machine Learning

Implemented models:

- Logistic Regression
- Random Forest
- XGBoost

Evaluation metrics:

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
